
# 🎉 Emoji Randomizer Project  

A fun and interactive web app that generates random emojis with colors each time you click.  

## 🔧 Tech Stack  
- HTML  
- CSS  
- JavaScript  

## 🚀 Features  
✔️ Random emoji generator  
✔️ Dynamic colors  
✔️ Lightweight and responsive
